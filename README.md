# Resume Website

This is the source code for my professional resume website, hosted on GitHub Pages.

## Features
- **Mobile Responsive:** Uses the Jekyll 'Minimal' theme.
- **Markdown Based:** Easy to update via `index.md`.
- **SEO Optimized:** Custom titles and descriptions in `_config.yml`.

## How to host your own
1. Fork this repository.
2. Update `index.md` with your information.
3. Enable GitHub Pages in your repository settings (Settings > Pages > Source: Deploy from a branch > main).
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.
